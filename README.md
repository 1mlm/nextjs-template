# nextjs-template

- [`Next.js`](https://nextjs.org/) 16 with Turbopack
- [`TypeScript`](https://www.typescriptlang.org/)
- [`React`](https://react.dev/) 19
- [`Tailwind CSS`](https://tailwindcss.com/) 4 with custom properties & `light-dark()` theming
- [`shadcn`](https://ui.shadcn.com/), customized with [`Create`](https://ui.shadcn.com/init?base=radix&style=nova&baseColor=neutral&theme=violet&iconLibrary=hugeicons&font=outfit&menuAccent=subtle&menuColor=default&radius=medium&template=next&rtl=false): Radix primitives, Nova style, **Violet theme**, Neutral base, Medium radius, [`Outfit`](https://fonts.google.com/specimen/Outfit) font
- [`Hugeicons`](https://hugeicons.com/)
- [`Biome`](https://biomejs.dev/) with TailwindCSS directive parsing
- Organized folder structure: `/shadcn/{ui,cpns,lib,hooks}`, `/styles/`
